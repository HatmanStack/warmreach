# Backend API Reference

Auto-generated reference for the Python Lambda shared-services layer
(`backend/lambdas/shared/python/shared_services/`). One page per module is
generated from the module, class, and function docstrings at build time.

Regenerate with:

```bash
npm run docs:api:py
```

This is generated output. Edit the docstrings in the source modules, not these
pages. The canonical list of services and their purpose also lives in
`docs/ARCHITECTURE.md` and `CLAUDE.md`.
